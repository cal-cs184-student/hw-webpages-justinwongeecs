Task 1: Bezier Curves with 1D de Casteljau Subdivision

How does de Casteljau's algorithm work?

  • Given \(n+1\) control points and a value \(t\) for parameterization, we generate \(n\) new points along the line segments between each point by lerping two consecutive points based on \(t\).
  • We use this helper lerp function to get a value between a and b parameterized by \(t\). inline float lerp(float a, float b, float t) { return (1.0f - t) * a + t * b; }
  • The crux of the algorithm is recursion. We recursively generate \(n\) new points from \(n+1\) control points until we are left with one point which is our evaluated point on the beizer curve, parameterized by \(t\).
